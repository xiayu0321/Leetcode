## 二叉树的后序遍历（简单）

### 题目

给你一棵二叉树的根节点 `root` ，返回其节点值的 **后序遍历** 。

**提示：**

- 树中节点的数目在范围 `[0, 100]` 内
- `-100 <= Node.val <= 100`

链接：https://leetcode.cn/problems/binary-tree-postorder-traversal/

### 思路

二叉树的后续遍历是先访问左节点->右节点->根节点，对于到达每个子树时仍按照这个方式进行遍历，因此可以考虑使用递归方式。首先确定递归退出条件，即当节点没有时（root == None）直接退出，然后按照输出顺序递归左子树和右子树，最后将根节点的值加入到 res 中。

另一种方式是栈方式，利用栈显式实现递归过程。只要根节点或栈存在，首先将所有左子树的 val 存储在栈中，直到找到最左节点，在栈中弹出该节点，此时判断该节点是否有右子树或者该节点的右节点是否已经访问过（用 pre 来记录添加到 res 的节点）。如果不存在右节点或者该右节点已经被访问过，则该节点直接加入到最终 res 中，并且 将 pre 指向该节点，表明该节点已被访问，同时将该节点设为空节点，防止重复访问该节点。如果该节点的右节点存在并且该右节点没被访问，则将该节点重新加入到栈中，然后将该节点指向该右节点。再重复进行，直至root 为空并且栈为空。
