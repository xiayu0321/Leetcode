## 2的幂（简单）

### 题目

给你一个整数 n，请你判断该整数是否是 2 的幂次方。如果是，返回 true ；否则，返回 false 。

如果存在一个整数 x 使得 n == 2x ，则认为 n 是 2 的幂次方。

提示：

- $-2^{31} <= n <= 2{31} - 1$

链接：https://leetcode.cn/problems/power-of-two

### 思路

2的幂指的是数字能连续被2整除，直至元素变为1。因此首先想到循环，判断整数 n 是否能被2整除，如果取余运算不为0，则返回 False，否则就进行整除。直至整数n 为1。注意负数和0的情况，此时都为 False。

如果不采用循环的方法，此时就要使用位运算，2的幂在位运算中可表示为整数 n 转化为二进制时只包含一个1。因此，可以找到二进制数中1的个数进行判断或者直接去掉最低位的1再去判断余下的数是否为0。

技巧：直接将 n的二进制表示的最低位 1 移除的方法：
**n & (n - 1) 和n & (-n)**
